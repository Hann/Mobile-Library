* 탭을 구성할 수 있다.

  • 초기 선택되는 탭은 nDefaultIndex 옵션을 통해 조절 할수 있다.

<다양한 모양의 탭을 구성할 수 있도록 다음과 같은 CSS 클래스를 제공한다.>

[sClassPrefix]base : 탭컴포넌트를 감싸고 있는 최상위 class
[sClassPrefix]tab-cont : 탭을 감싸고 있는 박스의 class
[sClassPrefix]tab : 탭 class
[sClassPrefix]taba : 탭 속의 태그 class
[sClassPrefix]panel-cont : 패널을 감싸고 있는 박스의 class
[sClassPrefix]panel : 패널 class
[sClassPrefix]selected : 선택된 탭 또는 패널 class (disply:block 속성을 추가한다)
