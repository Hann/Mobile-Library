* 한 페이지에 다 수의 Scroll 컴포넌트를 배치하여 구성할 수 있다.

  • 수평 스크롤은 bUseHScroll 옵션을 통해 조절가능하다.

    수직 스크롤은 bUseVScroll 옵션을 통해 조절가능하다.

    가속도 여부는 bUseMomentum 옵션을 통해 조절가능하다.

    가속시 감소계수는 nDeceleration 옵션을 통해 조절가능하다.

    스크롤바의 표기 여부는 bUseScrollbar 옵션을 통해 조절가능하다.

    모바일기기회전시 스크롤영역 자동 갱신은 bAutoResize 옵션을 통해 조절가능하다.

    스크롤 되는 고정영역의 Width값은 nWidth 옵션을 통해 조절가능하다.

    스크롤 되는 고정영역의 Height값은 nHeight 옵션을 통해 조절가능하다.



1. 가로 목록 : content가 가로 방향으로 많아 가로방향으로 가속 스크롤한다.

<기본 옵션을 제외하고, 추가로 설정한 옵션은 아래와 같다.>

{
bUseHScroll : true,
bUseVScroll : false,
bUseMomentum : true,
nDeceleration : 0.0005,
nHeight : 91
}

위 옵션 사용시, 스크롤바가 있고, 좌,우로 가속(감속계수 0.0005)가능한 스크롤러를 사용할 수 있다.


2. 세로 목록 : content가 세로 방향으로 많아 세로방향으로 스크롤한다.

<기본 옵션을 제외하고, 추가로 설정한 옵션은 아래와 같다.>

{
bUseHScroll : false,
bUseVScroll : true,
bUseMomentum : false,
nHeight : 300
}

위 옵션 사용시, 스크롤바가 있고, 위,아래로 가속 불가능한 스크롤러를 사용할 수 있다.

<주의>

bUseMomentum 옵션이 true 일 경우,
기기사양과, content의 크기에 따라 스크롤 도중 잔상이 남는 오류가 발생할수 있으므로 적용시에 유의하기 바란다.
